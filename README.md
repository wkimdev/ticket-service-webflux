# ticket service with webflux
- 추후 회사 프로젝트를 위해, 서버단의 리액티브 자바 서비스를 제공해보자 만든 간단한 프로토타입 프로젝트.
- 티켓 예매 서비스 with webflux 

# tech stack

```
- webflux  
- java8
- mongodb-reactive
- junit 5
- lombok 
- jxjava2
```
